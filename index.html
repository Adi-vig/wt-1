Relevance of EDAI/DT Projects in VIT

  • Core Course-Based
  • Interdisciplinary
  • Industry-Oriented
  • Social Impact
  • Hackathons & Competitions
  • Example Projects

Core Course-Based Projects

These EDAI projects are aligned with your core subjects, like embedded systems, control theory, software engineering, etc. They help you apply theoretical concepts to real-world applications and are often evaluated by professors, making them crucial for academic performance and internal assessment.

Interdisciplinary Projects

These projects combine concepts from multiple branches like mechanical, electrical, computer science, and design. They teach collaborative development, broaden technical exposure, and are highly valuable in research and higher studies.

Industry-Oriented Projects

EDAI projects mentored or inspired by industry problems enhance employability. They often involve real constraints, documentation, and client interaction—giving you a pre-internship like experience within college itself.

Social Impact Projects

These projects target community problems like waste management, assistive tech, or rural automation. They demonstrate social responsibility, boost CVs for social fellowships, and often receive funding and recognition.

Hackathons & Competitions

Many EDAI teams participate in inter-college or national innovation challenges. These not only build credibility but also bring networking opportunities, internship offers, and occasionally, startup ideas!

Example EDAI Projects

  • Smart Dustbin Uses ultrasonic sensor and servo motor to open lid when someone approaches. (Core + Social Impact)
  • Gesture-Controlled Robot Controlling a robot wirelessly using hand gestures and accelerometer. (Core + Interdisciplinary)
  • Smart Receptionist System Face recognition with ESP32, IoT-based door control, and web dashboard. (Interdisciplinary + Industry-Oriented)
  • PDF Knowledge Extractor using LLM + RAG A full-stack web app to intelligently answer questions from large PDFs. (Core + Research)
  • Drone vs Bird Classification using Micro-Doppler Radar ML-based classification using radar signatures. (Research + Industry)
  • Automatic Change Detection in Satellite Images Detecting urban development using SAR data and Google Earth Engine. (Research + Social Impact)
  • Smart Parking System ESP32 + sensors + web dashboard for booking and live monitoring. (Industry + IoT)
  • Maze-solving Robot using SLAM ROS2-based robot with DFS navigation and live mapping. (Core + Research)
  • Compiler Front-End Lex/Yacc-based mini C compiler with TAC generation. (Core Course Project)
  • Health Monitoring Wearable Tracks vitals and sends data to a web interface. (Interdisciplinary + Social Impact)
